Crittool

Crittool is a CRUD web app that facilitates standardized online critiques for artists and designers. The UI/UX is crafted to encourage objective feedback by structuring the critique process around the fundamentals of art and design.

The project makes use of asynchronous javascript, cloud-hosted databases, image analysis via RESTful API, and server-side coding in the Node.js environment.

Utilizes: JavaScript, HTML5/CSS3, VueJS, Ajax, Cloud Vision API, Firebase: Authorization, Cloud Functions, Cloud Firestore (NoSQL), Cloud Storage, Cloud Hosting

Bladers Field Guide

Bladers Field Guide is a responsive web app built with React. It displays information for inline skating tricks, including photos and descriptions. It features a clean UI/UX that lets users look through different tricks on any device.

I developed this project because I love to skate and want to share it! The sport needs more young people with an understanding of the trick vocabulary. This app provides a quick way to learn about different tricks.

Utilizes: JavaScript, HTML5/CSS3, React

Ready or Not

Ready or Not is a project that explores the commonality of our fears. Through color, motion, and typography, the site creates a calming atmosphere. This atmosphere encourages more honest interaction.

I developed this site to learn more about MySQL databases and experiment with color and motion.

Utilizes: JavaScript, HTML5/CSS3, MySQL
